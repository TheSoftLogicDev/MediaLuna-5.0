# MediaLuna Music Player v5.0 - MP5 EDITION 

O MediaLuna é um reprodutor de mídia universal baseado na estética clássica do Windows 7 (MPC-BE / Windows Media Player), focado em privacidade e versatilidade. Ele é capaz de processar arquivos locais sem a necessidade de upload para servidores.

##  Funções Principais
- **Suporte** a MP3, MP4, OGG, WAV, PNG, JPG, GIF, TXT, MD e HTML.
- **Engine MP5:** Decodificação de arquivos no formato `.mp5`.
- **Controles Avançados:** Shuffle, Repeat One, Repeat All e saltos de 10 segundos.
- **Adicionar Diretório:** Carregue pastas inteiras de uma vez.
- **MP5 Factory:** Ferramenta integrada para converter seus arquivos comuns em ".mp5".

## Como usar
1. Baixe o arquivo 'index.html'.
2. Abra em qualquer navegador moderno.
3. Arraste seus arquivos ou use os botões "+ Arq" ou "+ Pasta".

## Estética
Interface baseada no windows 7
